# Robotics Learning Platform

## Project Description
This project is a web platform developed using Laravel to manage robotics courses for a robotics school. The platform allows administrators, teachers, and students to access courses and manage educational robotics kits.

## ER Diagram
The database includes the following entities:

- Users
- Roles
- Groups
- Courses
- Robotics Kits
- Group Course

Each user has a role and students belong to a group. Groups can have multiple courses and each course uses a robotics kit.
